PBP Marketplace

I created the PBP marketplace webapp as a side project to learn more about JSON Web Tokens (JWTs), Unitesting, Deployement Pipelines and API Documentation.

My goal was to be able to tie all these pieces together to create a decoupled architecture where the frontend is built in Angular and the backend is built in Python (Flask) and all communication between the two is facilited by an API.

Breaking down the tech stack

The core idea for the stack I chose was one where the technology did not stand in the way of my exploration of the topics I wished to explore.

For the back end, Python (flask) was perfect. You only need about 9 lines of code to get started. From there you can take the app in any direction you wish to. This barebones nature of flask allowed me to build my own structure and that was exactly what I amimed for. app can be as simple as for this project because I wanted to explore setting up the project in an unopinionated environment that allowed me to build from barebones.

For the frontend, I picked Angular simply because I am familiar with it and setting it up would be a breeze.
